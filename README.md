# Tech Store Purchase System

## Description

The Tech Store Purchase System is a C++ console-based application that simulates a simple shopping checkout process for a technology store.

The program allows customers to select products from a catalogue, enter quantities, calculate their purchase subtotal, apply discounts, and display the final amount payable.

This project demonstrates fundamental C++ programming concepts including arrays, loops, conditional statements, user input handling, and mathematical calculations.

## Features

- Customer name input
- Display technology product catalogue
- Select multiple products
- Enter product quantities
- Calculate purchase subtotal
- Automatically apply discounts
- Display final purchase summary

## Technologies Used

- C++
- Standard Template Library (iostream, iomanip)
- Console Application
